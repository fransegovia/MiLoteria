# MiLoteria
Guarda todos los números a los que juegas a la lotería, añade información del dinero que juegas a cada número o si lo compartes con alguien.

# Características
Con esta aplicación podrás guardar todos los números a los que juegas a la lotería de Navidad (u otro sorteo), añadiendo información como el dinero que juegas a cada número o si lo compartes con alguien. 

Será muy fácil ver todos los números que juegas, cuánto te has gastado y con quién juegas o compartes cada uno de los números.

Nunca has tenido tanto control de lo que juegas en el sorteo de la lotería!!!

# Compatibilidad
La aplicación es compatible con cualquier dispositivo iOS o iPadOS a partir de la versión 16, también puede ser instalado en macOS con M1 o superior.

# Política de Privacidad
MiLoteria no recoge ninguna información personal ni de ningún tipo, toda la información de las loterías se guarda localmente en el dispositivo del usuario.
